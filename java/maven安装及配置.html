    maven官网下载地址http://maven.apache.org/download.cgi

    下载解压完maven之后配置环境

    变量名：MAVEN_HOME     变量值：F:\maven\apache-maven-3.0.5

    在cmd输入mvn -v      如果出现版本信息则表示安装配置成功

    本地仓库存储配置
    打开maven安装目录,备份并打开conf的settings.xml

    修改<localRepository>键值对包含的路径,可参考以下配置

    <localRepository>F:\taotao\repository</localRepository>

    最后在<mirrors>键值对添加以下信息

         <mirror>
          <id>nexus</id>
          <mirrorOf>*</mirrorOf>
          <name>Human Readable Name for this Mirror.</name>
          <url>http://127.0.0.1:8080/nexus-2.7.0-06/content/groups/public/</url>
        </mirror>

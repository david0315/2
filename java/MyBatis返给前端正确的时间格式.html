    前台获取位时间戳，后端解决办法之一

        问题描述：前端获取后台接口返回的数据，时间是long类型的时间戳而不是时间类型2019-09-25 17:07:32

        项目： JAVA web

        工具：eclipse版本Mars.2 Release (4.5.2)

        语言：Java SQL

        架构：SSM，Maven项目

        后端解决方法：SQL语句中添加DATE_FORMAT(APPLY_TIME, '%Y-%m-%d %H:%i:%s') AS APPLY_TIME

        

    select 
  	PROBLEM_ID,
	SET_KEY,
	SERIAL_NO,
	PROBLEM_TYPE,
	PROBLEM_DESC,
	DEAL_FLAG,
	APPLYER,
	APPL_DATE,
	DATE_FORMAT(APPLY_TIME, '%Y-%m-%d %H:%i:%s') AS APPLY_TIME,
	APPLYER_ORG_CODE,
	APPLYER_ORG_NAME,
	REVIEWER,
	REVIEW_DATE,
	DATE_FORMAT(REVIEW_TIME, '%Y-%m-%d %H:%i:%s') AS REVIEW_TIME,
	DEAL_DESC
from bs_problem_info

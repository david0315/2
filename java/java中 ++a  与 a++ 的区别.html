    • java中 ++a 与 a++ 的区别
    原文出处:本文由博客园博主未来相濡以沫提供。
    原文连接:https://www.cnblogs.com/si-137/p/11764786.html

    public static void main(String[] args)
    {
    int a = 5;
    a ++;
    System.out.println(a);

    int b = 5;
    ++ b;
    System.out.println(b);

    // 像上面这种不参加运算是没有影响的


    int c = b ++ * 5;
    System.out.println(c);
    System.out.println(b);
    // ++在后面是先操作，再对自身加1

    int d = ++ b * 5;
    System.out.println(d);
    System.out.println(b);
    // ++在前面是先让自己加1，再执行操作

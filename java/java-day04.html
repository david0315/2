    IntelliJ快捷键

    　　导入包　　alt + enter

    　　删除光标所在行　　ctrl + y

    　　复制光标所在行　　ctrl + d

    　　格式代码　　ctrl + alt + l

    　　单行注释　　ctrl + /

    　　多行注释　　ctrl + shift + /

    　　自动生成代码　　alt + ins

    　　移动代码　　alt + shift + 上下按键

    定义方法的完整格式：

    　　修饰符 返回值类型 方法名称（参数）{

    　　　　方法体;

    　　　　return 返回值

    　　}

    return作用

    　　停止当前方法

    　　将后面的返回值给调用处

    方法的调用

    　　单独调用

    　　打印调用

    　　赋值调用

    定义方法的注意事项

    　　对于void方法没有返回值，只能单独使用，不能打印调用或赋值调用

    　　return只能有效返回一次，不然机器不知道要返回的是哪个值

    　　return的返回值类型和方法的返回值类型一致

    　　不能在return后面写代码

    方法重载

    　　多个方法名称一样，但参数的列表不一样

    　　好处

    　　　　只需要记住一个方法名称，就能使用类似功能

    方法重载下列因素不会报错

    　　参数个数不同

    　　参数类型不同

    　　参数多类型顺序不同

    方法重载下列因数会报错

    　　参数名称不同，但参数类型相同

    　　方法的返回值类型不同

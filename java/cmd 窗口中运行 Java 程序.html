    1、CMD

    　　　　命令提示符（Command Processor）（CMD）

    　　　　CMD命令：开始－>运行－>键入 cmd（在命令行里可以看到系统版本、文件系统版本）

    2、对文件夹操作的部分命令

    • 启动：win+R
    • 切换盘符：盘符名称：
    • 进入文件夹：cd 文件夹名称
    • 进入多级文件夹：cd 文件夹1\文件夹2\文件夹3——shift
    • 回到上一级：cd..
    • 回到盘符根路径：cd\
    • 查看内容：dir
    • 清屏：cls
    • 退出：exit

    3、cmd 窗口中运行 java 程序的过程解析（以 HelloWorld.java 文件为例）

    　　 
    public class HelloWorld{
        
        public static void main(String[] args){
            System.out.println("HelloWorld!!!");
        }
        
}
    HelloWorld.java 文件内全部代码

    　　2.1　javac HelloWorld.java

    　　　　　　编译源程序生成字节码（ Bytecoe ）文件。Java 编译器读取 Java 源程序并将其翻译成 Java 虚拟机（ Java Virtual Machine, JVM）

    　　　　能够理解的指令集合，且以字节码的形式保存在文件中。字节码文件以 .class作为扩展名。

    　　　　注：即生成了 HelloWorld.class 文件

    　　2.2　java HelloWorld

    　　　　　　运行字节码文件。Java 解释器读取字节码，取出指令并翻译成计算机能执行的代码，完成运行过程。

    　　　　注：即运行 HelloWorld.class 文件

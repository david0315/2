    　　 1、解压java linux包

            tar zxvf jdk-8u121-linux-x64.tar.gz

            2、配置环境变量

    　　sudo vi /etc/profile

           3、 添加如下信息：

    #Java Env
export JAVA_HOME=/home/ubuntu/build/java/jre1.8.0_231
export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
export PATH=$PATH:$JAVA_HOME/bin

           4、配置生效：

    　　source /etc/profile

           5、验证效果：

    　　

     

     　 

          备注小细节信息：

    　　当使用java -jar  demo.jar 时，此时关闭对话框，项目就会停止，如下处理修复：

    　　nohup java -jar XXX.jar >log.out &

          小细节②：window下查询特定端口和杀该端口对应的进行pid

    　　　　netstat -ano | findstr 443  //查询固定端口是否被占用

    　　　　tasklist | findstr 2976       //查询该线程被哪个进行名占用

    　　　　

     

     　　　 taskkill /pid 2976  -t  -f  杀掉该进行

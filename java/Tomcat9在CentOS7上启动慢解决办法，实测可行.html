    • Tomcat9在CentOS7上启动慢解决办法，实测可行
    原文出处:本文由博客园博主成少爷提供。
    原文连接:https://www.cnblogs.com/master-cheng/p/11883747.html

    1.修改jdk目录下的java.security文件

    vim /usr/lib/jvm/jdk1.8.0/jre/lib/security/java.security

    将如下配置
    securerandom.source=file:/dev/random

    改为
    securerandom.source=file:/dev/urandom

    2.修改tomcat9下conf的catalina.properties

    将tomcat.util.scan.StandardJarScanFilter.jarsToSkip=后面加载的jar包删除

    改为tomcat.util.scan.StandardJarScanFilter.jarsToSkip=*.jar

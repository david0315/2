    二、Myeclipse的使用

    1.在workspace中工作区中有一个文件夹.metadata：在该文件夹中存储了当前MyEclipse的工作状态

    2.将.metadata文件夹删除之后，下一次再次进入这个工作区的时候，是一个全新的开始，但是会发想IDE工具中，所有的项目都丢失了，没关系，这里只是丢失了myeclipse的项目，硬盘上不会针对的丢失

    3.windows+shou view:是用来显示我们误关闭的窗口

    4.切换布局，针对不同开发项目量身定制的窗口类型

    5.当布局被破坏之后，想要复原

    windows+reset

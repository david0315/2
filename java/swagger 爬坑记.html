    后来，经过2个小时的折磨，发现，第33行，

    @ApiModelProperty(name = "名称")

    这里，如果想使用swagger-ui.html进行测试的话，value必须传递. 修改成如下所示：

    @ApiModelProperty(value = "随便写")

    成功，一切正常。

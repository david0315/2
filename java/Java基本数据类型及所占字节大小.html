    一、Java基本数据类型

    　　基本数据类型有8种：byte、short、int、long、float、double、boolean、char

    　　分为4类：整数型、浮点型、布尔型、字符型。

    　　整数型：byte、short、int、long

    　　浮点型：float、double

    　　布尔型：boolean

    　　字符型：char

     

    二、各数据类型所占字节大小

    　　计算机的基本单位：bit .　　一个bit代表一个0或1

    　　byte：1byte = 8bit　　　　 1个字节是8个bit

    　　short：2byte

    　　int：4byte

    　　long：8byte

    　　float：4byte

    　　double：8byte

    　　boolean：1byte

    　　char：2byte

    java企业级开发基本工具

    一、开发必备工具

    1、Java开发环境

    • JDK （官网下载即可，需要注册账户)
    • JRE

    2、Java企业编码开发工具

    • IntelliJ IDEA(建议使用Idea，插件使用更加完善，界面更加美观)
    • Eclipse 

    3、关系型数据库选择

    • mysql(企业运用最多)
    • oracle(功能比较强大)
    • sqlserver(初学者)。
    • 推荐使用连接工具navicate Premium

    4、目前使用的非关系型数据库

    • mongoDb(推荐使用工具 NoSQLBooster for MongoDB 功能比较强大),
    • Redis（RedisDesktopManager客户端显示工具）,
    • MQ （官网客户端）;

    5、接口调试工具

    • PostMan (自测调用接口反馈的数据，和调用接口的响应时间)
    • 花生壳 （与前端远程接口调用测试，存在网络延迟问题，属于小问题），TeamViewer （比较专业的远程工具），向日葵

    6、服务器

    • tomcat  (必须会，常用)
    • weblogic
    • 服务器类型很多，各取所需

    7、服务器部署工具

    • Xftp (可视化界面)

    • Xshell （黑窗口命令行操作）

    8、虚拟机学习工具

    • VMware
    • 镜像文件下载
    • Linux命令(用于对服务器进行操作)

    9、英语查询工具（英语不好必备，我就是必备的，哈哈）

    • 有道翻译

    10、脑图（用于理清思路和流程）

    • XMind

    11、记录学习中的笔记

    • 有道云笔记（用于记录账户和代码中遇到的问题，用于巩固）

    12、对前端需要了解的知识

    • HTML,CSS,JS(基本知识，需要有所了解)
    • VUE,Angular (前后端分离前端使用的框架)
    • 等

    　　这是我工作半年中，在工作中用到的工具，合理的使用这些工具，提高程序开发的进度，让自己有足够充分的时间做更多自己想做的事情，如果觉得需要补充的可以留言评论。

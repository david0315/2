    前言

    Java文件的运行过程：

    1，javac.exe：编译器

    2，java.exe：解释器

    微软shell下运行实例：

    C:\Users\Administrator>cd D:\文档\JAVALIAnxi\

C:\Users\Administrator>d:

D:\文档\JAVALIAnxi>javac HelloWorld.java

D:\文档\JAVALIAnxi>java HelloWorld
hello java

    （1）先切换到文件所在的目录；

    （2）编译文件；

    （3）解释文件，输出文件包含的内容；

    （ps：更推荐大家使用IntelliJ IDEA来编译Java程序）

     

    使用Java输出“hello world”

    （这是一个“HelloWorld.java”文件内容（文件名最好不要带空格，否则会出错））

    程序：

    public class HelloWorld {
    public static void main (String [] args) {
        System.out.println ("hello world");
    }
}

    "//"：单行注释

     

    1.第一行的第三个单词必须与本文件的名字一致，即“HelloWorld”就是文件名字，"public class"代表“类”，“HelloWorld”就相当于是类的名称(类是java源代码的基本组织单位)。

    public class HelloWorld

     

    2.第二行内容是固定的，代表main方法，标志着程序的起点。

        public static void main (String [] args)

     

    3.第三行代表则程序内容的输出，希望输出什么内容就在“println”后边的括号“（""）”内填什么。

            System.out.println ("hello world");

     

    以上便是Java的简单程序结构。

    1.如何下载及打开nexus

    nexus下载地址：http://www.sonatype.org/nexus/downloads/

    下载war包暂时还不清楚怎么操作的，但是有一位网友提供了一个版本的下载地址https://sonatype-download.global.ssl.fastly.net/nexus/oss/nexus-2.12.0-01.war

    --浏览器输入http://127.0.0.1:8080/nexus-2.7.0-06/显示错误，无法连接到服务器
    原因是我们没有配置好tomcat

    只需将下载的war包放置到tomcat的webapps文件夹里，并启动tomcat即可

     

    --下载的版本是nexus-2.14.1-01-bundle.zip

    解压后配置环境变量：

    NEXUS_HOME=F:\nexus\nexus-2.14.5-02\bin

    在PATH中加入;%NEXUS_HOME%

    --Nexus Bundle默认端口是8081；故需在浏览器输入http://localhost:8081/nexus/

    为方便启动，可将nexus配置成服务

    a.在控制台输入nexus install即可将nexus配置成服务

    b.启动nexus start

    定义：

    a、有static 修饰的类方法和静态方法；

    b、由static修饰的变量为静态变量或者类变量；

    c、静态方法和静态变量都是由类在磁盘中被创建产生的，与类同时存在，同时消亡；

    d、静态变量又称为成员变量，在类中是全局变量，可以被类中的所有方法调用。、

    e、静态变量有jvm(Java虚拟机)自动初始化，以下是各种类型的值所对应的初始化的变量；

    整型----0;

    浮点型------0.0；

    字符串类型 ----null;

    字符----0(unicode码为0的字符，提示：该字符不可见);

    布尔类型（Boolean）--------false

    使用

    1、使用static声明的属性：被static声明的属性将被称为全局属性；

    2、使用static声明的方法：可以直接调用类名；

    比如 你创建了一个类Person;

    class Person{

    public  int id;

    public String name;

    public Person(){

    }

    public  static void  shuxing(){

    system.out.println("这是一个静态方法，直接调用类名.方法就可以调用该方法！");

    }

    //这是空参构造器

    然后在main方法中直接调用类名就可以实现；

    Person.shuxing();//这样就可以实现了！

    }

    3 注意：使用static方法的时候，只能访问static的属性和方法，非静态的属性和方法是不可以访问的。

    public int id;

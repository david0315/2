    这是进入博客园第20天，思前想后，才决定开通博客。

    所谓的思前想后，其实说白了就是瞻前顾后，畏手畏脚，怕这怕那，说到底，好像懂在怕什么，好像又不懂。

    既然选择开通了博客，就说明要写一些东西，写什么呢？

    本人目前从事软件测试工作，最基础的黑盒测试，接触过不少语言，其实好多都是会写个"Hello World!"，涉入不深，每一次都是一时兴起，坚持不了几天，就会放弃。

     1 package com.liming.world;
 2 
 3 /**
 4  * @Author: LiMing

 7  * @File: HelloWorld
 8  */
 9 public class HelloWorld {
10     public static void main(String[] args) {
11         System.out.println("Hello World!");
12     }
13 }

     

    我讨厌这样的自己，非常的讨厌，有心想去改变，可行动起来的时候，又不知如何开始，从哪里开始改变。

    如今27的年纪，也马上奔三了，今年刚有一个女儿，说起来很幸福。可真的幸福么？女儿跟着母亲在老家，而我在上海，一千多里的距离，好远，好远....

    好像是悲观派，又非要说自己是乐观派....

    好了就说到这边吧，说出来也好多了，明天又是美好的一天...

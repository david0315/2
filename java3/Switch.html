    switch(expr1){

    case a:{

    System.out.println("a");

    break;

    }

    case b:{

    System.out.println("b");

    }

    case c:{

    System.out.println("c");

    break;

    }

    }

    其中 expr1只能时整数表达式或枚举类型,整数表达式:int基本 类型或int包装类Integer 其byte

    short,char可以隐含转换为int,所以也可以是是它们和其包装类,long 和 String类型不行,但是在jdk1.7以后可以使用String

    break关键字在switch选择一个case代码块执行完毕后跳出,如果没有break,它将执行下一行代码而不跳出switch

    上述 expr1=a 输出为:a

             expr1=b 输出为: b

                                        c

             expr1=c 输出为:c

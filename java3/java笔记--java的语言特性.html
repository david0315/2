    java的语言特性

    1.简单性：例如：java不再支持多继承，而c++是支持多继承的，多继承比较复杂。

                                c++中有指针，java中屏蔽了指针的概念。所以相对来说比较简单。

                                 //注：java语言的底层是c++，不是c语言。

    2.安全性

    3.可移植性：java语言可以做到一次编译，到处运行。

    4.面向对象：java是纯面向对象的，更符合人类的思维。

    5健壮性：和自动垃圾回收机制有关，简称GC机制，java语言运行过程中产生的垃圾不用手动清理

    今日所学：

    构造方法没有返回值（即return为空）。

    this：实例（对象）的引用。

    JVM：①static方法区：存静态数据

    　　   ②栈区：引用

    　　   ③堆区：对象/对象内存地址 ----> new 类名() ;

    public > protected > package(default) > private

    封装性 ----> 访问控制权限

    以private修饰时，get/set方法可以提供接口访问private。

    如：

    private int a ;
get方法：public int getA () {
            return a ;
        }
set方法：public void setA (int a) {
            this.a = a ;
        }
取用：Demo demo = new Demo () ;
         demo.setA(2) ;
         int a = demo.getA() ;
     System.out.println(a) ;

    首句：package：包定义

    　　　import：导入相关类

    内部类：在类的内部定义类。

    内部类优点：可以方便的使用外部类的属性。

    内部类缺点：破环类的基本结构，因此要慎用。

    Outer outer = new Outer() ;  //实例化外部类对象
Outer.Inner inner = outer.new Inner() ;  //实例化内部类对象

    代码块：

    ①普通代码块：即{ }括起来的。

    ②构造块：在构造方法之前执行，每执行一个构造方法都会执行一次。作用是补充构造方法。直接写在类里。

    ③静态代码块：static { } （注：不加分号）最先执行，且只执行一次。

    匿名对象/类：new 类名() ;

    上下移动行（一行或多行）快捷键：alt+上下方向键

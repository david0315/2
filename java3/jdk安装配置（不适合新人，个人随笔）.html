    • jdk安装配置（不适合新人，个人随笔）
    原文出处:本文由博客园博主急于入门提供。
    原文连接:https://www.cnblogs.com/dzgg/p/11996919.html

     

    jdk下载：https://www.oracle.com/technetwork/java/javase/downloads/index.html 

    按个人喜好下载相应版本，一般比最新的低一个版本

    下载下来后进行安装

    先出现jdk安装位置，个人根据情况选地方

    然后出现的是jre的安装位置，最好跟jdk安装在一个文件夹，但不能放在jdk包下

    安装完成配置环境变量：右键我的电脑 选择属性，高级系统设置

     新建变量：CLASSPATH

    变量值：.;%JAVA_HOME%\lib;%JAVA_HOME%\lib\tools.jar（注意最前面那个点“.”）一定要复制全

    新建变量：JAVA_HOME

    变量值：就是你安装jdk的位置

    找到系统自带变量path，最后加入这一段：

    ;%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;
    配置完后，检查是否成功：运行CMD，输入 java没有错误的话 输入 javac
    输入 java -version 可以查看版本号

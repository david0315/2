    Java集合框架 Collection

    Collection是一个接口

    步骤 1 : Collection

    Collection是 Set List Queue和 Deque的接口
    Queue: 先进先出队列
    Deque: 双向链表

    注：Collection和Map之间没有关系，Collection是放一个一个对象的，Map 是放键值对的
    注：Deque 继承 Queue,间接的继承了 Collection

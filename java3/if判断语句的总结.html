    • if判断语句的总结
    原文出处:本文由博客园博主AP-TX4869提供。
    原文连接:https://www.cnblogs.com/japtx/p/11784845.html

    1、表达式：关系表达式或逻辑表达式；

    2、表达式的运算结果应该是“真”或者“假”；

          真：执行该语句；

               假：跳过该语句，执行下一条语句；

    3、“语句”可以是单语句也可以是复合语句；

    4、else if 可以有多条，没有上限；

    5、if 和 else if都需要判断表达式真假，else则不需要，else if 和 else必须配合if一起使用，不能单独使用。

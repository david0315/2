    一、新建一个maven web工程

    step1

    　　File --》 New --》 Project

     

     

     

    step2

    　　按下图步骤操作

     

     

     

    step3

    　　填写项目信息

     

     

     

    step4

    　　选择本地的maven安装环境。若不熟悉maven，可以参考 https://www.cnblogs.com/l-y-h/p/11147939.html

     

     

     

    step5

    　　创建项目名以及项目保存的位置

     

     

     

    step6

    　　等待maven自动联网下载即可。可能会没有 java，resource等文件夹（手动创建）。

     

     

     

    step7

    　　手动创建java，resource等文件夹。选择File --》 Project Structure

     

     

    step8

    　　选择Modules，找到项目 右击 选择 New Folder。

     

     

    step9

    　　新建一个文件夹java，右击选择Sources即可。

     

     

     

     

    step10

    　　同理创建resource，此时选择Resources，选择OK即可。

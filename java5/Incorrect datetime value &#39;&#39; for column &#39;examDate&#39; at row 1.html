    • Incorrect datetime value '' for column 'examDate' at row 1
    原文出处:本文由博客园博主听惊雷提供。
    原文连接:https://www.cnblogs.com/share-record/p/11584040.html

    出问题的程序：
    user.setCreateTime(new Date());

     

    控制台图片一张，问题是：Incorrect datetime value: '' for column 'createTime' at row 1

     

    解决方案：

    import java.text.SimpleDateFormat;
import java.util.Date;
import java.sql.Timestamp;
//取当前时间
Date nowdate=new Date();
//转换时间格式
SimpleDateFormat simpleDate = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss"); 
user.setCreateTime(Timestamp.valueOf(simpleDate.format(nowdate)));

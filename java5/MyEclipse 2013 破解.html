    1、运行cracker.jar

    用 cd 命令找到 cracker.jar所在目录
    然后运行命令 java -jar cracker.jar

     

    在Usercode 那一栏随便输入你的用户名，并选择你的 Myeclipse 10 相应版本，然后点击SystemId…

     

     

    生成 SystemId ，然后点击 Active ，会自动生成 License key、Activation code 和 Activation key：

    然后点击菜单 Tools -> 2.SaveProperties，将上一步生成的 License key、Activation code 和 Activation key 保存到 C:\Documents and
    Settings\Administrator.myeclipse.properties 中。

    点击菜单 Tools -> 1.ReplaceJarFile…，然后选择到 My eclipse 安装目录下的\Common\plugins 目录，点击“打开”，然后等待，直到出现替换文件成功信息。这就表示破解完了。

    Mycat是一个数据库中间件，并不能直接当为数据库使用，它的所有操作数据来源都是对应数据库中取出来的。所以我们要安装运行Mycat需要有对应的数据库，这里预先安装了MySQL5.5，总共安装了3台服务器，对应三个数据库。

    Mycat是java语言开发的，运行Mycat我们需要先安装JDK环境，并且需要JDK1.7以上的版本才可以。

    先查看linux的JDK

    java -version

    • 建议安装JDK1.8的环境，安装教程：https://blog.csdn.net/xlecho/article/details/97266591

    下载Mycat Linux安装包

    下载地址：https://github.com/MyCATApache/Mycat-download/tree/master/1.6-RELEASE

    下载完成之后移动到自动以目录

    // 自定义一个目录
mkdir /usr/local/Mycat

// 将上传的压缩包移动到Mycat文件
mv Mycat-server-1.6-RELEASE-20161028204710-linux.tar.gz /usr/local/Mycat/

    使用tar命令解压

    tar -xzvf Mycat-server-1.6-RELEASE-20161028204710-linux.tar.gz

    尝试启动

    // 进入bin目录，输入如下命令（启动）
./mycat start

// 进入bin目录，输入如下命令（停止）
./mycat stop

    如果没有报错那就已经正确安装了。

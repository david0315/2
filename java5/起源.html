    day one

    计算机的基本功能操作包括以下四个方面：

    逻辑运算：与、或、非；

    算术运算：加、减、乘、除；

    数据比较：大于、小于、等于、不等于、大于等于、小于等于；

    数据传送：输入、输出、赋值。

    JAVA 严格区分大小写

    Java 是一种简单的、面向对象的、分布式的、解释的、安全的、可移植 的、性能优异的多线程语言。
    计算机只能识别0和1，通过编程语言常人看得懂的方法去告诉计算机怎么做。而计算机会通过编译器翻译得到1和0组成的命令。

    jre(java runtime environment),Java运行环境

    jvm(Java Virtual Machine),Java 虚拟机

    Java API(Java Application Programming Interface)Java 应用程序编程接口

    只要有jvm任何平台都可以执行java字节码程序

    源代码.java文件通过javac 编译成.class文件  最终通过jvm编译成程序

     

    简单了解对于程序员需要知道的计算机硬件工作原理：

    1. 主板　　支撑系统平台工作的功能
    2. CPU　　中央处理器处理接受到的指令，系统的运算和控制核心，是信息处理、程序运行的最终执行单元
    3. 内存　　随机存取存储器。CPU和硬盘的数据桥梁。计算机中所有程序的运行都是在内存中进行
    4. 硬盘　　存储设备
